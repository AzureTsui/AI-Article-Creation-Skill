## 📖 这是什么？

一份从 **12 篇爆款深度文章** 中提炼出的 AI 创作 Skill 系统提示词范式。

它不是万能提示词模板，而是一套**写作灵魂指南** —— 让 AI 不再写出"首先……其次……综上所述"的八股文，而是像一个**在大厂踩过坑、替读者算过账、敢跟全网对线的产品经理**一样，写出有温度、有态度、有交付物的硬核内容。

## 🚀 如何使用

### 作为 AI 系统提示词

将 [`AI_article_creat_skill.md`](./AI_article_creat_skill.md) 的内容直接用作 AI 对话的 System Prompt 或 Custom Instructions，驱动 AI 以"锦维斯风格"进行文章创作。

### 作为写作参考手册

按照 **创作 SOP（6 Phase Checklist）** 逐步检查你的每一篇文章，确保：

1. ✅ 选题具备反共识角度或杀手级痛点
2. ✅ 匹配了 7 种骨架之一
3. ✅ 前置投入了 20% 定义目标/框架/风格
4. ✅ 包含 ≥3 处降维比喻 + ≥2 处情绪化表达
5. ✅ 结尾提供开箱即用的交付物
6. ✅ 全文零八股文词汇
